MongoDB

What is MongoDB?

MongoDB is a document database that leverages the concept of key value pairs, and it does not have a fixed schema. It is part of a larger family of databases known as NoSQL.

Why MongoDB?

MongoDB offers many advantages over traditional databases, such as:

  • Speed - MongoDB can operations up to 50x faster than SQL based databases.
  • Scalability (Horizontal) - MongoDB allows for linear scaling as the data is spread across many machines.
  • High Flexibility - MongoDB supports various data types.
  • Sharding - MongoDB allows you to partition your data across multiple nodes.
  • Well documented - MongoDB has a comprehensive set of documentation.
  • A lot of unstructured data which cannot fit in relational databases - MongoDB can handle large amounts of unstructured data without compromising performance.

Applications of MongoDB in Data Science?

MongoDB can be used in many areas of data science such as real time customer analytics and IoT.

What to Learn in MongoDB for Data Science?

For data science, the key areas you should get familar with is the basics, pymongo and MQL.
